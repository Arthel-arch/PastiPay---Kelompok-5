# PastiPay---Kelompok-5
**Proyek UTS Web Client Development - Universitas Cakrawala.**
Website profil resmi PastiPay, sebuah platform payment gateway terkemuka untuk bisnis online di Indonesia. Proyek ini menyajikan informasi layanan ekosistem pembayaran yang aman, cepat, dan transparan, lengkap dengan visualisasi arsitektur sistem dan dasbor interaktif yang responsif.

## ✨ Fitur Utama yang Ditampilkan:
* **Comprehensive Payment Ecosystem:** Integrasi Bank Transfer, E-Wallet, QRIS, dan Minimarket.
* **Built for Builders:** Halaman dokumentasi REST API, API Sandbox, dan SDK/Plugins untuk developer.
* **Transparent Economics:** Komparasi biaya transaksi dan keunggulan kompetitif (PastiPay vs Kompetitor).
* **Bank-Grade Security:** Edukasi standar keamanan infrastruktur (Bank Indonesia Standard, AES-256, PCI DSS).
